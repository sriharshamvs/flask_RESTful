# Flask RESTful API

## Deployment Process

- Clone the Repo 
```
git clone git@github.com:sriharshamvs/flask_RESTful.git
cd flask_RESTful
```

- Install Dependencies
```
pip3 install -r requirements.txt
```

- Create DB
```
python3 helpers/create_tables.py
```
This will create a data.db in the project root directory.

- Run the Application
```
python3 app.py
```
